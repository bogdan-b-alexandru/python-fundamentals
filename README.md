# Python Fundamentals

Practice exercises built while transitioning from Java to Python, 
as part of a self-directed full-stack AI development curriculum.

## What I practiced
- Python class syntax coming from a Java background
- Instance variables and the `self` parameter
- Lists and the `.append()` method
- F-strings for string formatting
- Dictionaries for structured data
- Functions with multiple return values
- List comprehensions
- Error handling with try/except

## Files
- `student.py` — OOP class with enrollment methods
- `job_posting.py` — Dictionary structure for job data
- `functions.py` — analyze_job function with multiple return values
- `list_practice.py` — List comprehension filtering and transformation
- `error_handling.py` — Safe skill lookup with IndexError handling

## Part of a larger curriculum
This is Project 1 of 5 leading to a full-stack AI agent app 
built with Python, Flask, PostgreSQL, React, and the Claude API.
